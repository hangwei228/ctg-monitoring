# 功能模块 Functional Modules

本目录包含信号预处理流程的**核心功能模块**，每个子目录是一个可独立调用的算法组件。相比 `ctuuhb/` 和 `private_dataset/` 里的完整流程脚本，这里粒度更细，便于复用与替换。

## 模块清单

| 模块 | 作用 | 对应完整流程 |
|---|---|---|
| `quality_scoring/` | 基于 Q 值的信号质量评分与筛选（阈值 0.6） | 第 1 步 |
| `evt_denoise/` | EVT 软阈值去噪 | 第 2 步 |
| `standardize_split/` | 信号标准化 S(t) = F(t) − B(t) 与 70/15/15 数据集划分 | 第 5、8 步 |

## 与完整流程的关系

`ctuuhb/` 和 `private_dataset/` 里的脚本会串联调用这些模块。需要复现完整流程，可直接使用两个目录下的脚本；需要单独使用某算法，可在此使用对应模块。

## 使用说明

每个模块的输入输出、参数说明、调用示例请查看其子目录下的 README.md。

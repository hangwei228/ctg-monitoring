# Signal Preprocessing 信号预处理

本目录包含 CTG 胎心监护信号的预处理代码。

## 代码组织

分为三个部分：

- `ctuuhb/`：针对 **CTU-UHB 公开数据集**（PhysioNet）的**完整预处理流程**，含二分类与三分类版本
- `private_dataset/`：针对**私有数据集**的**完整预处理流程**，含二分类版本
- `modules/`：可**独立调用的功能模块**，是完整流程中核心算法的拆解，公共 / 私有数据集共用

## 处理流程

两套完整脚本使用相同的核心方法，差异在数据格式与标签解析：

1. Q 值筛选（阈值 0.6）
2. FHR / UC 信号清洗
3. SG 滤波 + EMD + K-Means 基线提取
4. 标准化：S(t) = F(t) - B(t)
5. 滑动窗口数据增强
6. 长度统一为 900 点
7. 70 / 15 / 15 分层划分（train / val / test）

## 使用说明

- 完整预处理 → 进 `ctuuhb/` 或 `private_dataset/`，看里面的 README
- 单独用某个算法模块（比如只做去噪） → 进 `modules/`，看对应子目录的 README

 ## 参数说明与调整建议 Parameter Configuration

本项目参数在自建数据集上调优得到。**如需应用到其他数据集，请根据实际信号质量、采样率、数据规模自行调整。**

### 参数位置

具体参数在各 py 文件的开头部分定义，修改时请**同时保持二分类和三分类版本的参数一致**，以避免结果偏差。

## 标签标注方法 Label Annotation

本数据集的分类标签依据以下规范生成：

- **判读标准**：《妇产科学》（第 9 版，人民卫生出版社）
- **判读流程**：由 **3 名产科医师独立判读**，仅采用 **三人判读结果一致** 的样本纳入数据集
- **分类粒度**：
  - 二分类：正常 / 异常
  - 三分类：正常 / 可疑 / 异常

未达成三人一致的样本不纳入训练数据，以保证标签质量。
